# 重症心身障害

## 重要: 行政上の分類

**重度の知的障害 + 重度の肢体不自由** を合併する状態。医学診断ではなく **行政・福祉分類**。

## 大島分類（大島の分類表）

縦軸に運動機能（走れる〜寝たきり）、横軸にIQを配置した25分類:

| 区分 | 該当 |
|---|---|
| **1〜4** | 重症心身障害（IQ35以下 かつ 運動機能 座位以下） |
| 5〜9 | 周辺児（重症心身障害に近い） |
| 10〜25 | 重症心身障害非該当 |

## 背景疾患

- 脳性麻痺
- 染色体異常症（ダウン症以外の重症例）
- 周産期障害・低酸素脳症
- 先天性代謝異常
- 髄膜炎・脳炎後遺症
- てんかん性脳症

## 主要特性

### 医学的状態
- 重度運動障害（寝たきり〜座位維持）
- 重度知的障害
- **医療的ケア** 必要頻度が高い:
  - 経管栄養（胃瘻・経鼻）
  - 喀痰吸引
  - 気管切開・人工呼吸器
  - 導尿
- てんかん合併 高頻度
- 側彎・拘縮
- 呼吸障害
- 嚥下障害・誤嚥性肺炎

### コミュニケーション
- 発語なし、または発語非常に限定的
- 表情・視線・体動でのコミュニケーション
- 介助者の読み取り能力が重要

## 生活機能への影響（ICF観点）

- **心身機能**: 運動、知的、呼吸、嚥下等ほぼ全領域
- **活動**: 移動、セルフケア、コミュニケーション
- **参加**: 24時間介護下での参加

## 「医療的ケア児者」との関係

2021年 **医療的ケア児支援法** 制定。医療的ケアを要する児者は重症心身障害と重なる部分が多いが、知的障害を伴わない医ケア児（例: 人工呼吸器管理の知的正常児）も含む。

## 計画相談でのポイント

### 1. 医療連携が必須

- 主治医・訪問診療医・訪問看護との密な連携
- 医療的ケアを担える福祉事業所の確保が最大の課題
- **医療型** の福祉サービス（医療型短期入所・医療型児童発達支援）

### 2. 家族介護の重み

- 24時間介護、特に母の負担極大
- レスパイトの定期化（[[66_Services/短期入所]]）は精神保健上も必須
- きょうだい児への配慮

### 3. 教育

- 特別支援学校（肢体不自由・病弱）
- 訪問教育
- 医療的ケアが必要な通学には看護師配置が必要

### 4. 移行期（18歳問題）

- 児童期サービス → 成人期サービス への切替
- 医療は小児科 → 内科への移行（成人移行支援）

### 5. 親亡き後の最難課題

- 重症心身障害者GHの絶対的不足
- **施設入所支援** が現実的選択肢となる場合も
- 本人の尊厳を守りながらの選択が問われる
