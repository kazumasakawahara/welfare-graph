# 意思決定支援を踏まえた後見事務のガイドライン

## 概要

成年後見人・保佐人・補助人（**後見人等**）が意思決定支援の視点で後見事務を行うための実務指針。令和2年、成年後見制度利用促進における **多機関合意** で発出。

## 背景

- **障害者権利条約12条**（法律の前の平等）: 後見人が本人の意思を代替するのではなく、意思決定を支援する
- **成年後見制度利用促進基本計画**（第一期）でパラダイム転換
- 従来の「代行決定中心」から「意思決定支援中心」への転換

## 基本的考え方

### 1. 後見人等は意思決定支援の担い手

- 代行決定（本人に代わって決める）から **意思決定支援**（本人が決められるよう支える）へ
- 民法858条 **意思尊重義務** の具体化

### 2. プロセスの重視

- 結果だけでなく **意思決定のプロセス** を重視
- 本人参加の機会確保

### 3. チームアプローチ

- 後見人等単独ではなく、**支援チーム** で
- 福祉・医療関係者との協働
- [[64_Methods/意思決定支援会議]] の活用

## 意思決定支援の基本原則

### 原則①: 本人への支援

- 自己決定の尊重
- 本人を意思決定の主体として位置付け
- 必要な情報・選択肢を分かりやすく提供

### 原則②: 意思及び選好の推定

- 本人が表明困難な場合、過去の言動・選好から推定
- **推定** と **最善の利益判断** を区別

### 原則③: 最善の利益

- 意思・選好の推定も困難な **最後の手段**
- 本人の人生観・価値観に最大限沿う

## 後見事務における具体的適用

### 1. 身上保護

- 施設入所・退所の判断
- 医療受診・入院の判断
- 介護サービス利用の判断

**原則**: 本人の意思確認 → 推定 → 最善の利益 の順。

### 2. 財産管理

- 日常金銭: 本人が使える仕組みを優先（代行ではなく）
- 重要財産処分: 本人の意思を最大限尊重、合意形成プロセスを記録

### 3. 居所指定

- 「後見人等が決める」のではなく **本人の希望を基点**
- やむを得ず本人の希望と異なる判断をする場合、根拠を記録

## 意思決定支援のプロセス（8ステップ）

1. **本人の意思決定能力のアセスメント**（重要: 能力は流動的）
2. **意思形成支援**: 情報提供・選択肢の明確化
3. **意思表明支援**: 表明しやすい環境
4. **意思実現支援**: 表明された意思の実現
5. **意思決定支援チームの編成**
6. **意思決定支援会議の開催**
7. **決定と記録**
8. **モニタリング**

## 計画相談専門員との協働

### 連携のポイント

- 後見人等 × 計画相談 × 福祉事業所 の **三者協働**
- 情報共有の仕組み（ケース会議の定期化）
- 役割分担の明確化

### 本vaultの方針

README に記載の通り:
- 成年後見は **支援者の輪の一員**、中心ではない
- 金銭管理の第一選択は **日常生活自立支援事業**（社協）
- 必要時に成年後見へ

計画相談専門員はこの見立ての中で、後見人等と協働する。

## 派生論点

### 特定の選択場面での留意

- **入院同意**: 後見人等に医療同意権はない（解釈論）
- **選挙権**: 令和元年から被後見人も回復
- **財産処分の方向性**: 本人の QOL 維持を基準
