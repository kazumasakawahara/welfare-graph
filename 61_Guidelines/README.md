# 61_Guidelines — ガイドライン・手引・通知

厚労省・自治体・関連団体が発する運用基準。法令の具体運用を示す。

## 対象範囲
- サービス等利用計画作成サポートブック
- 意思決定支援ガイドライン（厚労省）
- 身体拘束適正化の手引
- 障害者虐待防止マニュアル
- 強度行動障害支援者養成研修テキスト
- モニタリング標準期間通知
- 地域移行支援の手引
- 地域定着支援の手引
- 報酬改定Q&A（最新年度）
- 障害児通所支援の手引
- 就労支援ガイドブック
- 成年後見制度利用促進基本計画

## 命名規則
`61_Guidelines/{略称}.md`

## 必須 frontmatter
`issuer`（発行元）/ `source_url` / `version` / `effective_date` / `review_due` は **必須**。

## 運用
- 原文PDFは `raw/guidelines/` に保存
- 「記録要件」「研修要件」「周期要件」は チェックリスト形式で抽出
- 自治体独自ガイドラインは `issuer` に自治体名を明記
