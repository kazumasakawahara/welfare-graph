# ストレングスモデル

## 概要

米国 **Charles Rapp** らが精神障害者の地域生活支援のために開発した支援モデル。**「問題ではなく強みに着目する」** が核心。

日本では精神保健福祉領域を中心に、相談支援全般に浸透。計画相談の基礎的支援姿勢として広く活用。

## 6つの原則

### 1. 精神障害者はリカバリーし、生活を改善し、向上させることができる
- 「治らない」「変わらない」という思い込みを超える

### 2. 焦点は病理ではなく個人の強みに
- 診断名・問題リストから、能力・興味・願望へ視点を転換

### 3. 地域を資源のオアシスと捉える
- 施設サービスだけでなく、地域の人・場所・機関・制度を資源として活用

### 4. 利用者が支援過程の指揮者である
- 支援者が決めるのではなく、本人が方向性を決める
- 「本人中心」の徹底

### 5. 支援者と利用者の関係は基本であり本質である
- 信頼関係こそが支援の基盤
- ラポール構築に時間をかける

### 6. 支援の場は地域である
- オフィス・施設ではなく、本人の生活の場へ出向く
- アウトリーチの重視

## 4つの技術要素

### 1. エンゲージメント
信頼関係の構築。時間をかけて本人を知る。

### 2. ストレングスアセスメント

**9領域** で本人の強み・願望・資源を把握:

| 領域 | 内容 |
|---|---|
| 日常生活 | 生活スキル・好み |
| 経済 | 収入・金銭管理 |
| 仕事・教育 | 職歴・学習歴・意欲 |
| 余暇・レクリエーション | 趣味・楽しみ |
| 人間関係 | 家族・友人・支援者 |
| 健康 | 体調・医療 |
| 住環境 | 住居・地域 |
| 精神性・霊性 | 信念・価値観 |
| その他 | 独自の領域 |

### 3. 個別計画
本人の願望から出発した個別支援計画。ゴールは本人の言葉で。

### 4. 資源の獲得
地域の人・場所・機関・制度を繋ぎ、本人の願望実現を支える。

## 従来モデルとの対比

| 領域 | 従来（問題モデル） | ストレングスモデル |
|---|---|---|
| 起点 | 問題・症状 | 強み・願望 |
| 役割 | 専門家が治療 | 本人が指揮、支援者は伴走 |
| 場所 | 病院・施設 | 地域 |
| 資源 | サービス中心 | 地域資源の総動員 |
| 時間感覚 | 診療時間内 | 生活時間に合わせる |

## リカバリー概念との関係

- リカバリー = **「障害があっても意味ある人生を生きる」**
- 医学的治癒（cure）ではなく、生き方の回復
- ストレングスモデルはリカバリーの支援論

## 知的障害者支援への適用

### 適用可能性

- 精神障害領域発祥だが、知的障害にも応用可
- 「できない所を訓練で埋める」から「強みで補う」への転換
- [[66_Services/計画相談支援]] の基本姿勢

### 具体的適用

1. **願望の聞き取り**: 「将来何をしたい？」「好きなことは？」
2. **強みの発見**: 日常の小さな「できる」を発見・言語化
3. **地域資源の活用**: 特別支援事業所だけでなく、地域の趣味サークル・カフェ・図書館 等
4. **本人の言葉での計画**: 専門用語ではなく本人に意味のある言葉で

### 注意

- 「強みを見つけられない」と支援者が感じる時こそ、自分の見方を疑う
- 家族が「できない」と思っていることも、環境調整で「できる」になる
