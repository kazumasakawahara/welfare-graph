# ICF（国際生活機能分類）

## 概要

**WHO が2001年に採択した障害・健康の国際分類**。従来の ICIDH（障害＝機能障害+能力低下+社会的不利）の医学モデルを脱し、**生活機能モデル** へ転換。日本語訳は厚労省が公表。

計画相談業務の **アセスメントの基本フレーム** として広く活用。

## 生活機能の3層

```
┌─────────────────────────────────┐
│  心身機能・身体構造              │  Body Functions & Structures
├─────────────────────────────────┤
│  活動                            │  Activities
├─────────────────────────────────┤
│  参加                            │  Participation
└─────────────────────────────────┘
       ↕ 相互作用
  背景因子（環境因子・個人因子）
```

### 心身機能・身体構造（Body Functions / Structures）
- 生理的・心理的機能
- 身体の解剖学的部分
- 例: 知能機能・感覚機能・運動機能・身体部位

### 活動（Activities）
- 個人によって行われる課題や行為
- **能力**（標準環境下で可能な最大）と **実行**（実生活の実情）で評価
- 例: 学習、日常生活動作、コミュニケーション

### 参加（Participation）
- 生活・人生場面への関わり
- 例: 就労、地域社会参加、家族生活

## 背景因子（Contextual Factors）

### 環境因子
- 人的環境（家族・支援者）
- 物的環境（住居・道具）
- 社会的環境（制度・サービス）
- 態度（偏見・理解）

### 個人因子
- 年齢・性別
- 生活歴・学歴
- 性格・価値観
- 対処スタイル

## ICFモデルの核心: 相互作用

従来（医学モデル）: 「障害があるから困る」
ICF（生活機能モデル）: 「心身機能×環境×個人 の相互作用で生活機能が決まる」

→ 心身機能を「治す」だけでなく、**環境を変える** ことで生活機能は改善可能。

## 支援現場での適用

### 1. アセスメントの基盤

- [[65_Assessments/ICF評価]] で構造化
- 「何ができるか（能力）」と「何をしているか（実行）」を分離
- 環境因子・個人因子の影響を可視化

### 2. 計画相談での活用

- [[66_Services/計画相談支援]] の **ニーズ整理** に ICF
- 「心身機能の問題」だけでなく「環境調整で変わる部分」を明示
- 本人の強み・資源を可視化

### 3. 多職種での共通言語

- 医療・福祉・教育・労働で共通分類
- 障害者権利条約も ICF を基盤
- 記録・連携のフォーマット

## ICFを使う際のコツ

### 本人中心で記述

- 「できないこと」より「何ができるか」
- 「困っていること」だけでなく「本人の願い」

### 環境因子を見逃さない

- 「性格の問題」に見える事象が環境要因だったり
- 改善可能な環境要因を列挙

### 実行と能力のギャップ

- 「家ではできる」vs「事業所ではしない」
- 環境調整の手がかり

### 時系列で捉える

- 一時点の状態像ではなく経過の中で
- ライフステージで変わるニーズ

## 限界・批判

- 項目が膨大で実務運用が難しい（簡易版が各国で作成）
- 個人因子のコード化が未整備
- 測定尺度ではなくあくまで分類体系
