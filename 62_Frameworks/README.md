# 62_Frameworks — 理論・フレームワーク

支援の思想的・理論的バックボーン。どの技法を選ぶかの判断基準を与える。

## 対象範囲
- ICF（国際生活機能分類）
- ストレングスモデル
- パーソンセンタードプランニング
- ケースマネジメント
- エンパワメントアプローチ
- エコロジカルモデル
- トラウマインフォームドケア（TIC）
- 愛着理論
- QOLモデル
- レジリエンス理論
- リカバリーモデル（精神領域）
- バイオサイコソーシャルモデル
- アドボカシー概念

## 命名規則
`62_Frameworks/{理論名}.md`

## 必須 frontmatter
`origin`（提唱者・団体）/ `year`（提唱年）/ `domain`（適用領域）。

## 運用
- 理論↔支援技法（`64_Methods/`）のリンクは `informs` / `underpins` を使う
- 一次文献URLは `source_url` に記載（書籍の場合はISBN）
- 対立・補完関係の理論は `contradicts` / `complements` でリンク
