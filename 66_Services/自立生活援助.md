# 自立生活援助

## サービスの目的

障害者支援施設・GH等から **地域生活（一人暮らし）へ移行** した者に対し、一定期間、**定期的な訪問** 等により生活上の課題への相談・助言を行う。

## 対象者

- 障害者支援施設・[[66_Services/共同生活援助]]・精神科病院等から退所・退院して **一人暮らしを開始** した者
- 原則 **6か月以内** に開始
- **1年間** が原則（必要に応じ更新可）

## 支援内容

- **定期訪問**（週1回程度）
- **随時対応**（電話・訪問）
- 生活リズム・金銭管理・服薬・食事・対人関係の相談
- 関係機関との連絡調整

## 計画相談との接続

典型的な流れ:
```
地域移行支援（施設・病院で準備）
  → [[66_Services/共同生活援助]]（GH 一定期間）
  → 自立生活援助（一人暮らし初期1年）
  → [[66_Services/地域定着支援]]（24時間連絡体制・長期）
```

## 報酬

- 月額基本報酬（定期訪問頻度に応じた区分）
- 随時対応加算
