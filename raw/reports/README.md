# raw/reports/

学会発表・調査報告書・白書などの原本を保管。
取り込み先は **`30_Insights/`** または **`90_Meta/`**（メタ情報の場合）。

## 命名規則

```
raw/reports/{年月}_{発行元}_{タイトル略}.pdf
```

例:
- `raw/reports/2024-07_最高裁_旧優生保護法違憲判決全文.pdf`
- `raw/reports/2025_厚労白書_障害者福祉.pdf`
- `raw/reports/2024_日本相談支援専門員協会_調査報告.pdf`
