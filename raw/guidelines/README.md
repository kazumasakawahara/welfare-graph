# raw/guidelines/

厚労省・自治体・関連団体のガイドライン・通知・手引の原本を保管。
取り込み先は **`61_Guidelines/`**。

## 命名規則

```
raw/guidelines/{ガイドライン略称}_{版名}.pdf
```

例:
- `raw/guidelines/意思決定支援ガイドライン_平成29年.pdf`
- `raw/guidelines/身体拘束適正化の手引_令和4年度.pdf`
- `raw/guidelines/モニタリング標準期間通知_令和6年改定.pdf`

## 取り込み時の確認項目

ガイドライン取り込み時、wiki ページの frontmatter には以下を必須記入:

- `issuer`: 発行元（厚生労働省 / 自治体名 等）
- `source_url`: 公式 URL
- `version`: 版名（例: 「令和4年度版」）
- `effective_date`: 施行日
- `review_due`: 次回見直し予定日（3 年後を目安）
