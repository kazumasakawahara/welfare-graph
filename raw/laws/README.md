# raw/laws/

法令の **原文PDF・一次資料** を保存する場所。`60_Laws/` の各ページは、この raw/ 内の原文を要約したもの。

## 命名規則

```
raw/laws/{法令略称}_{版名}.pdf
```

例:
- `raw/laws/障害者総合支援法_令和6年改正.pdf`
- `raw/laws/障害者虐待防止法.pdf`
- `raw/laws/障害者差別解消法.pdf`
- `raw/laws/精神保健福祉法.pdf`
- `raw/laws/発達障害者支援法.pdf`
- `raw/laws/成年後見制度関係法令集.pdf`

## 運用

1. 法令改正時は旧版を `raw/laws/archive/` に移動、新版を配置
2. PDFは **加工せず原本** を保存
3. 抜粋引用する場合は `60_Laws/` のページ内で出典（条番号・ページ）を明記
