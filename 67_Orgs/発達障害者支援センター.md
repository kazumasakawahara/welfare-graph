# 発達障害者支援センター

## 機能・役割

発達障害の総合的支援を担う専門機関。都道府県・政令市に設置。**生涯を通じた一貫支援** を旨とする。

## 法的位置づけ

- [[60_Laws/発達障害者支援法]] 14条
- 設置主体: 都道府県・政令市
- 運営: 直営または社会福祉法人等への委託
- 全国に **100か所以上**

## 主要業務

### 1. 相談支援

- **発達障害の相談受付**（本人・家族・関係者）
- 乳幼児期〜成人期までの幅広い相談
- 電話相談・来所相談・訪問相談

### 2. 発達支援

- アセスメント・検査
- 個別支援計画作成補助
- 家族への助言

### 3. 就労支援

- 就労相談
- 職場環境調整への助言
- 就労系サービスとの連携

### 4. 普及啓発・研修

- 支援者向け研修
- 一般向け啓発（発達障害啓発週間 4月2-8日等）
- 教材・マニュアル開発

### 5. 関係機関連携

- 医療・保健・福祉・教育・労働の **ハブ機能**
- 自治体内関係課との調整
- 事例検討会

## 配置

- 所長
- **発達障害支援専門員**（社会福祉士・精神保健福祉士・心理職等）
- 必要に応じ 医師（嘱託）・作業療法士・言語聴覚士

## ライフステージ別支援の接続

| 時期 | 主な連携先 |
|---|---|
| 乳幼児期 | 保健センター・[[児童発達支援]]事業所 |
| 学齢期 | 学校・特別支援教育・[[放課後等デイサービス]] |
| 青年期 | ハローワーク・[[就労移行支援]] |
| 成人期 | [[計画相談支援]]・[[就労継続支援B型]]・医療機関 |
| 中高年期 | 高齢者福祉・認知症対応（ダウン症等） |

## 計画相談との連携

### 接続場面

1. **発達障害の診断がない／不明確** → 支援センターで見立て
2. **学齢期から成人期への移行** → 情報引継ぎ
3. **家族支援の強化** → 家族相談・家族教室
4. **事業所の対応困難** → 支援センターからの助言

### 役割分担

| 機能 | 支援センター | 計画相談 |
|---|---|---|
| 診断・見立て | ◎ | ー |
| 総合相談 | ◎（発達障害特化） | ○（総合） |
| サービス利用計画 | ー | ◎ |
| 継続モニタリング | ー | ◎ |
| 普及啓発 | ◎ | ー |

## 活用場面

### 計画相談専門員の事例相談

- 発達障害のある方の支援で困った時
- 対応方法のコンサルテーション
- 研修受講（発達障害の理解深化）

### 家族への紹介

- 診断後の家族支援
- 家族教室・ペアレントトレーニング
- ピアサポート

### 事業所への助言

- 放デイ・児童発達支援の支援方法
- 就労事業所の職場環境調整

## 課題

### アクセス格差

- 都道府県単位の設置のため、地方部ではアクセス困難
- 待機期間の長さ（新規相談数か月待ちも）

### 成人期支援の不足

- 児童期支援が中心になりがち
- 成人期の発達障害支援体制は発展途上

### 二次障害への対応

- 発達障害 + うつ・不安等の二次障害
- 精神科医療との連携強化が課題
