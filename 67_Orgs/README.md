# 67_Orgs — 関係機関（類型）

支援ネットワークを構成する機関の **類型** を収納。具体の事業所・医療機関（O-XXXX / M-XXXX）は `40_Stakeholders/` に入れる。

## 対象範囲
- 市町村障害福祉主管課
- 基幹相談支援センター
- 自立支援協議会
- 委託相談支援事業所
- 指定特定相談支援事業所
- 児童相談所
- 発達障害者支援センター
- 保健所
- 保健センター
- 精神保健福祉センター
- 障害者就業・生活支援センター（就ポツ）
- ハローワーク（専門援助部門）
- 地域障害者職業センター
- 特別支援学校
- 放課後児童クラブ
- 児童家庭支援センター
- 要保護児童対策地域協議会
- 家庭裁判所（成年後見・虐待）
- 警察（生活安全課）
- 社会福祉協議会
- 権利擁護センター
- 医療機関（精神科・発達外来・内科等）

## 命名規則
`67_Orgs/{機関類型名}.md`

## 40_Stakeholders/ との違い
| 層 | 粒度 | 例 |
|---|---|---|
| `67_Orgs/` | 類型（汎用） | `基幹相談支援センター.md` |
| `40_Stakeholders/` | 具体（仮名ID） | `O-0001.md`（利用者が使う具体事業所） |

`40_Stakeholders/` の各エントリから `67_Orgs/` の類型にリンク（`instance-of`）。

## 必須 frontmatter
`role`（役割概要）/ `mandate`（法的位置づけ）/ `referral_routes`（紹介経路）。

## 運用
- 「どんな時にどこへ繋ぐか」の意思決定支援に使える粒度で記述
- 類型↔サービス（`66_Services/`）は `delivers` でリンク
- 類型↔法令（`60_Laws/`）は `mandated-by` でリンク
