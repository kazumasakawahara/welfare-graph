# 基幹相談支援センター

## 機能・役割

地域における **相談支援体制の中核機関**。令和6年改正で強化された。

### 4つの中核機能

1. **総合的・専門的な相談支援**: 3障害（身体・知的・精神）＋発達障害＋難病の **総合相談**
2. **地域の相談支援体制の強化**: 委託相談・指定特定相談事業所への **バックアップ**
3. **地域移行・地域定着**: [[66_Services/地域移行支援]] / [[66_Services/地域定着支援]] の推進
4. **権利擁護・虐待防止**: 虐待防止の取組み・成年後見制度利用支援

## 法的位置づけ

- [[60_Laws/障害者総合支援法]] 77条の2（令和6年改正で強化）
- **市町村が設置主体**（直営または委託）
- 令和6年から設置が **努力義務から実質的義務へ**
- 自治体により委託法人（社協・NPO・社会福祉法人等）が運営

## 配置

- 相談支援専門員（主任相談支援専門員含む）
- 精神保健福祉士・社会福祉士
- 事務職員
- 令和6年度以降、**機能強化型** として常勤主任相談支援専門員配置で加算

## 主要業務

### 相談支援

- **総合相談窓口**: あらゆる障害種別・年代の相談受付
- **複合・困難ケース** への対応
- 計画相談事業所と協働

### バックアップ機能

- 指定特定相談支援事業所への助言・指導
- 研修開催
- スーパーバイズ（経験浅い相談支援専門員向け）

### 地域づくり

- [[67_Orgs/自立支援協議会]] の事務局・運営
- 地域の社会資源開発
- サービス事業所との関係構築

### 権利擁護

- 虐待の早期発見・対応（市町村との連携）
- [[66_Services/成年後見制度利用支援事業]] の推進
- 本人・家族の権利侵害相談

## 令和6年改正のポイント

基幹相談支援センターの機能強化:

1. **設置の実質的義務化**
2. **人員配置基準の明確化**
3. **機能強化加算の新設**（計画相談事業所のバックアップ実績）
4. **主任相談支援専門員の配置**
5. **地域の相談支援体制構築**の責任明確化

## 他機関との役割分担

### 市町村との関係

| 機能 | 市町村 | 基幹相談 |
|---|---|---|
| 支給決定 | ◎ | ー |
| 総合相談 | ○（窓口） | ◎（対応） |
| 虐待対応 | ◎（法定） | ○（協力） |
| 事業所指導 | ◎（監査） | ○（助言） |

### 指定特定相談支援事業所との関係

- 基幹相談 = 地域全体を見る
- 特定相談 = 個別の計画相談を担当
- 困難ケース・複合ケースは基幹に相談・連携

### [[67_Orgs/自立支援協議会]] との関係

- 基幹相談が協議会の事務局を担う自治体が多い
- 協議会で出た地域課題を基幹相談が受けて取組む

## 計画相談事業所が使う場面

### 困難ケース相談

- 他の制度・機関との複合課題
- 社会資源が見つからない
- 本人・家族との関係構築に困難
- 虐待疑い等

### 研修・スーパーバイズ

- 新人相談支援専門員の育成
- 事例検討会の場
- 主任相談支援専門員からの助言

### 地域資源開発

- 事業所不足を協議会・基幹に報告
- 新規事業所設立の相談
- 既存事業所の質向上

## 相談支援体制の3層構造

```
┌─────────────────────────────────┐
│ ② 基幹相談支援センター           │ ← 地域の中核
│   （市町村に原則1か所以上）      │
└──────┬──────────────────────────┘
       │ バックアップ
┌──────┴──────────────────────────┐
│ ③ 指定特定相談支援事業所         │ ← 個別ケース
│   （市町村に複数）                │
└──────┬──────────────────────────┘
       │ 計画相談・障害児相談
┌──────┴──────────────────────────┐
│ ① 本人・家族・サービス利用者     │
└──────────────────────────────────┘
```
