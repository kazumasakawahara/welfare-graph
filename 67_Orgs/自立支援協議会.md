# （地域）自立支援協議会

## 機能・役割

地域における障害福祉の **関係機関の集合協議体**。個別ケースの積み重ねから地域課題を抽出し、行政施策・社会資源開発に反映させる役割。

## 法的位置づけ

- [[60_Laws/障害者総合支援法]] 89条の3
- 市町村協議会・都道府県協議会の2層構造
- 市町村協議会は **市町村に設置努力義務**（実質ほぼ全市町村で設置）

## 協議会の構成

### 参加者

- 行政（市町村障害福祉課・関係課）
- [[67_Orgs/基幹相談支援センター]]（多くの場合事務局）
- 指定特定相談支援事業所
- サービス事業所（生活介護・就労継続・GH等）
- 医療機関
- 教育機関
- 雇用関係機関（ハローワーク・就ポツ）
- 当事者団体・家族会
- 民生委員・児童委員
- 社会福祉協議会
- 弁護士・司法書士（権利擁護）

### 階層構造

```
全体会（年1-2回）
  ├─ 運営会議（月1回程度）
  └─ 部会（テーマ別、月1回程度）
      ├─ 相談支援部会
      ├─ 就労支援部会
      ├─ 権利擁護部会
      ├─ 精神障害部会（にも包括）
      ├─ 児童部会
      ├─ 地域移行部会 等
```

## 主要な機能（5つ）

### 1. 個別の支援に関する情報共有

- 困難ケースの検討
- 事例検討会

### 2. 地域の関係機関によるネットワーク構築

- 顔の見える関係
- 連絡調整

### 3. 困難事例への対応のあり方に関する協議

- 対応事例の積み上げ
- 対応マニュアル・手順の整備

### 4. 地域の社会資源の開発

- 不足サービスの発見
- 新規事業所の設立支援
- インフォーマル資源の活用

### 5. 地域ケア体制の検証・評価

- 地域の障害福祉計画の検討
- 実施状況のモニタリング

## 令和6年改正での強化

- 自立支援協議会の運営・機能の **充実**
- 市町村の設置努力義務は維持（義務化は見送り）
- [[67_Orgs/基幹相談支援センター]] による運営支援の強化

## 計画相談事業所が関わる意義

### 地域の声を届ける

- 日々のケースから見える地域課題を協議会に
- 事業所不足・サービス制度の課題を行政に伝達

### 連携構築

- 他事業所・他機関との顔の見える関係
- 困難ケース時の協力体制

### 人材育成

- 事例検討会での学び
- 他事業所の実践から学ぶ

## 会議運営での課題

### ありがちな問題

- **形骸化**: 報告会に留まり実質協議なし
- **参加者の固定化**: 同じ顔ぶれで新しい視点が入らない
- **当事者不在**: 「当事者抜きの協議」になりがち

### 改善の視点

- ケースベース・課題ベースの協議
- 当事者・家族の参加促進
- アクションにつながる議論
- 小さな成功事例の共有
