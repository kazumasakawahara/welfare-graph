# 障害支援区分認定調査（80項目）

## 目的

障害福祉サービスの支給量決定の基礎となる **障害支援区分** を認定するための調査。「介護の手間」を客観的に評価するツール。

区分は **非該当 / 区分1〜6** の7段階。区分が高いほど必要な介護量が多い。

## 法的位置づけ

- [[60_Laws/障害者総合支援法]] 22条
- 実施主体: 市町村
- 認定調査員: 研修修了者（市町村職員または委託）

## 調査項目の構成（80項目）

### 1. 移動や動作等に関連する項目（12項目）
- 寝返り・起き上がり・座位保持
- 移乗・立ち上がり・両足での立位保持
- 片足での立位保持・歩行・移動
- えん下・食事摂取・排尿・排便

### 2. 身の回りの世話や日常生活等に関連する項目（16項目）
- 清潔の状況（口腔清潔・洗顔・洗髪等）
- 衣服の着脱
- 意思の伝達・説明の理解
- 金銭の管理・買い物
- 調理・掃除・洗濯・電話利用
- 交通手段の利用

### 3. 意思疎通等に関連する項目（6項目）
- 視力・聴力
- コミュニケーション
- 説明の理解
- 読み書き
- 感覚過敏・鈍麻

### 4. 行動障害に関連する項目（34項目）
大分類:
- 被害妄想的言動（12項目）
- 強迫観念的言動（8項目）
- **行動関連項目（12項目 / 重要）**: [[65_Assessments/行動関連項目]] 参照
- その他行動（2項目）

### 5. 特別な医療に関連する項目（12項目）
- 気管切開・呼吸器・酸素療法
- 経管栄養・IVH（中心静脈栄養）
- 透析・導尿・カテーテル
- 褥瘡処置・疼痛管理 等

## 認定プロセス

### 1. 申請
- 本人・家族が市町村に申請
- 新規 or 更新（原則3年毎）

### 2. 認定調査（80項目）
- 本人への面接調査（家族・支援者同席可）
- 自宅 or 事業所 or 病院等で実施
- 認定調査員は中立公正の姿勢

### 3. 医師意見書
- 主治医が作成
- 疾病・障害の医学的情報
- 特別な医療・生活上の留意事項

### 4. コンピュータ判定（一次判定）
- 80項目のスコアをコンピュータが分析
- 区分の候補を算出

### 5. 市町村審査会（二次判定）
- 認定調査結果 + 医師意見書 + 特記事項 を総合評価
- 最終的な区分決定
- 非該当・区分1〜6 の7段階

### 6. 区分認定の通知
- 本人・代理人に通知
- 有効期間: 原則3年（更新必要）
- 状態変化時は再認定申請可

## 区分と主要サービスの関係

| サービス | 必要区分 |
|---|---|
| [[66_Services/居宅介護]] | 区分1以上（通院等介助は区分2以上） |
| [[66_Services/重度訪問介護]] | 区分4以上+二肢麻痺等 or 行動関連10点以上 |
| [[66_Services/同行援護]] | アセスメント票基準 |
| [[66_Services/行動援護]] | 区分3以上+行動関連10点以上 |
| [[66_Services/生活介護]] | 区分3以上（50歳以上は2以上） |
| [[66_Services/短期入所]] | 区分1以上 |
| 重度障害者等包括支援 | 区分6 |
| [[66_Services/施設入所支援]] | 区分4以上（50歳以上は3以上） |
| [[66_Services/療養介護]] | 区分5-6 |

※ [[66_Services/就労継続支援B型]] / [[66_Services/共同生活援助]] / [[66_Services/就労移行支援]] 等は区分認定不要。

## 計画相談での活用

### アセスメント情報として

- 区分そのものだけでなく、80項目の **個別スコア** を参照
- どの項目で高得点か → 重点支援領域
- 行動関連項目の合計点数 → [[66_Services/行動援護]] 適格性

### 支給決定プロセスとの連動

- 区分が決まってから計画案を作成（通常）
- 区分に見合うサービスの組合せ
- 区分では測りきれない生活課題も計画に組込む

### 更新時の確認

- 3年毎の更新が基本
- 状態が変わった時は早期に再認定申請を市町村に依頼
- 計画相談専門員が変化を察知したら自治体に相談

### よくある問題への対処

#### 区分が低く出すぎた
- 特記事項欄に生活実態を詳記
- 再調査依頼
- 審査会への異議申立

#### 本人が調査に協力できない
- 家族・支援者同席
- 日常の様子を代弁
- 状態像が見える場所（事業所等）で調査

#### 認定調査員の障害理解不足
- 調査前に本人の特性を情報提供
- 知的障害・発達障害の特性書類の添付
