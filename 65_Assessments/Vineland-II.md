# Vineland-II 適応行動尺度

## 概要

**米国 AGS Publishing（現 NCS Pearson）** が開発した適応行動の標準化評価尺度。日本語版は2014年。知的障害・発達障害の診断において **DSM-5-TR が推奨** する中心的尺度。

「適応行動」= 日常生活でその人が実際にしていること（能力ではなく実行）。

## 適応行動の評価

| 領域 | 下位領域 | 内容 |
|---|---|---|
| **コミュニケーション** | 受容言語 / 表出言語 / 読み書き | 聞く・話す・読む・書く |
| **日常生活スキル** | 身辺自立 / 家事 / 地域生活 | 食事・着替え・家事・買物 |
| **社会性** | 対人関係 / 遊び / コーピング | 他者との関わり方 |
| （運動スキル）※6歳以下 | 粗大運動 / 微細運動 | 歩く・走る・操作 |

合計スコアで **適応行動総合スコア**（平均100・SD15）。

### 任意の領域
- **不適応行動**（internalizing / externalizing / 重篤）

## 実施方法

### 半構造化面接

- 本人をよく知る **家族・支援者** への面接
- 「できる／できない」ではなく **「実際にしているか」** を聞く
- 日常場面の具体例を挙げさせる

### 所要時間

- 30-60分
- 初回は長めに
- 面接法のほかチェック式「調査員記入用紙」版もあり

## スコアの解釈

### 標準スコア

- **平均 100 / SD 15**（IQと同じ体系）
- 低値 = 適応困難
- 領域間のばらつきも重要

### 適応水準

| スコア | 水準 |
|---|---|
| 130以上 | 非常に高い |
| 115-129 | 中程度に高い |
| 86-114 | 適応的 |
| 71-85 | 中程度に低い |
| 70以下 | 低い（適応困難） |

### 知的障害診断との関係

DSM-5-TR の知的障害診断では:
- 知能検査（WAIS等）: IQ概ね70未満
- **適応行動評価（Vineland-II 等）: 概念的・社会的・実用的領域のいずれかで困難**

知能検査だけでなく **適応行動評価が不可欠** とされる。

## 強み

- **国際的標準化**: 多国籍データ
- **包括的領域**: 生活全般
- **年齢範囲**: 0-90歳（乳幼児〜高齢者まで）
- **実行重視**: 「実際にしているか」
- **支援計画への転換**: 具体的項目が計画立案に使える

## 日本での実施

### 実施機関

- 児童相談所
- 発達障害者支援センター
- 医療機関（小児神経科・精神科・児童精神科）
- 大学・研究機関
- 療育機関（一部）

### 費用
- マニュアル・記録用紙購入
- 公認心理師等の有資格者による実施

## 計画相談での活用

### 診断補助として

- 療育手帳判定（自治体により基準異なる）
- [[63_Disorders/知的障害]] の診断

### アセスメント情報として

- 支援が必要な具体領域（下位尺度別スコア）
- 強み（高得点領域）
- 時系列変化（定期再評価で）

### 支援計画への反映

- 低スコア領域 → 重点支援
- 高スコア領域 → 強みとして活用
- 個別目標の具体化

### 家族との共有

- 「できる」「できない」ではなく「実際にしている」の視点
- ポジティブな表現への変換
- 目標設定の共通言語

## 類似尺度との比較

| 尺度 | 特徴 |
|---|---|
| **Vineland-II** | 国際標準、0-90歳、DSM-5推奨 |
| **S-M社会生活能力検査** | 日本独自、児童向け（0-13歳） |
| **ABAS-3** | 米国尺度、Vineland より簡易 |
| **TTAP**（成人自閉症評価） | ASD成人向け |

日本では Vineland-II が最も推奨されるが、自治体・機関により S-M が継続使用される例も多い。

## 計画相談専門員の活用パターン

### 1. 診断前の本人・家族相談

- Vineland-II を実施できる機関への紹介
- 診断・評価の意義の説明

### 2. 既実施のスコアを読む

- 医療機関や児童相談所で実施済のスコアを参考に
- 領域別の強み・弱みを計画に反映

### 3. 再評価の提案

- 数年ごとの再評価で変化把握
- 支援効果の測定に活用
