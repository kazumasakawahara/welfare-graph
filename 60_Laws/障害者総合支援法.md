# 障害者総合支援法

## 概要

障害者・障害児が日常生活および社会生活を営むための支援を総合的に行うことを目的とする法律（旧：障害者自立支援法）。障害福祉サービスの中核的根拠法。

## 目的（1条）

- 基本的人権を享有する個人としての尊厳にふさわしい日常生活・社会生活
- 障害の有無にかかわらず国民が相互に人格と個性を尊重し安心して暮らせる地域社会
- 社会参加の機会の確保

## 主要条文サマリ

### 5条：定義
- 「障害福祉サービス」の範囲（介護給付・訓練等給付）
- 「障害者」の定義（身体・知的・精神・難病）
- 各サービスの内容定義

### 22条：支給決定
- 市町村による支給要否決定
- 障害支援区分の認定（6段階）
- 支給量の決定基準

### 77条：市町村地域生活支援事業
- 相談支援、コミュニケーション支援、日常生活用具給付、移動支援等

### 77条の2：基幹相談支援センター
- 地域の相談支援の中核機能
- 総合相談・権利擁護・地域移行/定着支援の強化

## サービス体系（5条ベース）

| 区分 | サービス |
|---|---|
| 介護給付 | 居宅介護 / 重度訪問介護 / 同行援護 / 行動援護 / 療養介護 / 生活介護 / 短期入所 / 重度障害者等包括支援 / 施設入所支援 |
| 訓練等給付 | 自立訓練 / 就労移行支援 / 就労継続A・B / 就労定着支援 / 自立生活援助 / 共同生活援助 |
| 相談支援 | 計画相談 / 地域移行 / 地域定着 / 基本相談 |
| 自立支援医療 | 更生医療 / 育成医療 / 精神通院医療 |
| 補装具 | 補装具費支給 |
| 地域生活支援事業 | 市町村必須事業・都道府県必須事業 |

## 介護保険との関係（65歳問題）

65歳到達時に介護保険が優先となるが、障害福祉サービス固有のサービス（行動援護・重度訪問介護等）は継続利用可能。共通サービス（居宅介護・生活介護等）は介護保険へ移行が原則だが、本人の状態や必要量を勘案し例外あり（平成19年事務連絡）。

## 支援現場での主要論点

- 支給決定プロセスの透明性（障害支援区分ありき vs ニーズベース）
- 65歳問題（介護保険優先原則の個別判断）
- 報酬改定（3年周期）による事業運営への影響
- 地域格差（市町村単独事業の有無）
