# 障害者総合支援法（平成30年改正版）

## この版の主要改正点（2018-04-01 施行）

### 新設サービス
- **自立生活援助**: 施設・GH 等から地域へ移行した者への定期巡回・随時相談支援
- **就労定着支援**: 就労 6 か月経過後の就労継続を支援
- **共生型サービス**: 介護保険・障害福祉の双方の指定が可能に

### 高齢障害者対応
- 65 歳以上で障害福祉サービスを継続利用する場合の利用者負担軽減（一定要件）

### 障害児支援
- 居宅訪問型児童発達支援の新設
- 保育所等訪問支援の対象拡大

## 平成30年改正版に基づく主要サービス一覧

（令和6年改正版で就労選択支援が追加される前の状態）

| 区分 | サービス |
|---|---|
| 介護給付 | 居宅介護 / 重度訪問介護 / 同行援護 / 行動援護 / 療養介護 / 生活介護 / 短期入所 / 重度障害者等包括支援 / 施設入所支援 |
| 訓練等給付 | 自立訓練 / 就労移行支援 / 就労継続A・B / **就労定着支援（新設）** / **自立生活援助（新設）** / 共同生活援助 |
| 相談支援 | 計画相談 / 地域移行 / 地域定着 / 基本相談 |

## 令和6年改正での主要変更点

- 就労選択支援の新設（→ [[60_Laws/障害者総合支援法#令和6年改正]] 参照）
- 基幹相談支援センターの機能強化（77条の2 改正）
- 入院中の意思疎通支援（重度訪問介護の入院中サービス継続）
- 共同生活援助における一人暮らし等への移行支援強化
