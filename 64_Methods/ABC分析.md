# ABC分析（3項随伴性分析）

## 概要

行動を **A（先行事象）- B（行動）- C（結果事象）** の3要素で観察・記録し、行動の **機能**（なぜその行動が起きるか）を明らかにする手法。

気になる行動を「本人の性格」ではなく「環境との相互作用」として捉え直すための土台。

## 3要素

| 要素 | 内容 | 記録のポイント |
|---|---|---|
| **A (Antecedent)** | 行動の直前に何があったか | 誰が／どこで／何を言われ／前に何をしていたか |
| **B (Behavior)** | 観察可能な具体的行動 | 「暴れた」ではなく「腕を掴み引き寄せた」と具体化 |
| **C (Consequence)** | 行動の直後に何が起きたか | 周囲の反応・環境変化 |

## 行動の4機能（仮説生成）

ABCを集めると、行動が **何を得ているか／避けているか** が見える:

| 機能 | 例 |
|---|---|
| **要求獲得** | 欲しい物・活動を得る |
| **回避・逃避** | 嫌な課題・場面から逃れる |
| **注目獲得** | 支援者・周囲の注目を得る |
| **感覚刺激（自動強化）** | 自己刺激としての行動 |

## 記録シート例

```
日時: 2026-04-20 14:30
場所: 生活介護事業所 作業室

A (先行): 作業終了予告の30分前、終了時刻を間違えて伝えた
B (行動): 机上の部品を床に投げ、立ち上がって歩き回った（約5分）
C (結果): 職員が部屋を別室に移動、休憩を提案した

考察（仮説）: 時間変更の混乱＋不明瞭な説明 → 本人がパニック
             部屋移動で「嫌な課題から離れられる」→ 負の強化が働いた可能性
```

## 使い方の手順

1. **記録（最低3日〜2週間）**: 日常場面での収集
2. **パターン抽出**: 共通するA・Cを探す
3. **機能仮説**: 4機能のどれか、複合か
4. **介入計画**:
   - 先行事象の調整（A）
   - 代替行動の教示（機能は同じで適切な方法）
   - 結果事象の調整（C）
5. **効果測定**: 同じABC記録で行動頻度の変化を確認

## 禁忌・注意

- **「悪い行動を減らす」発想からの脱却**: 機能を満たす代替行動を教えないと、別の問題行動に置き換わる
- **個人の内面の推論に偏らない**: 「不満だった」ではなく観察可能な事象を記録
- **短期記録の過信**: 少数事例から機能を断定しない
- **支援者間の記録ばらつき**: 「暴れた」「パニック」等の主観用語を避ける訓練が必要

## 現場Tips

- **シンプルなフォーム** から始める（A3・1週間分）
- スマートフォンの記録アプリ活用（音声入力）
- **ケース会議の題材** として記録を共有 → 複数視点の仮説生成
- ポジティブな ABC 記録（うまくいった場面）も同じ手法で
