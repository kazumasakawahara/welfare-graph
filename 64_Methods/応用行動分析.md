# 応用行動分析（ABA）

## 概要

行動分析学を応用し、**学習の原理を使って社会的に意味のある行動変容** を目指す体系的アプローチ。自閉スペクトラム症の早期介入で最も研究されてきた療法の一つ。

## 基本原理

### 行動の随伴性（3項随伴性）

| A (Antecedent) | B (Behavior) | C (Consequence) |
|---|---|---|
| 先行事象 | 行動 | 結果 |
| 環境・指示 | 対象行動 | 強化／弱化 |

→ [[64_Methods/ABC分析]] で詳述

### 強化と弱化

- **正の強化**: 好ましい結果 → 行動増加
- **負の強化**: 嫌な結果の除去 → 行動増加
- **正の弱化**: 嫌な結果 → 行動減少（罰、倫理的配慮要）
- **負の弱化**: 好ましい物の除去 → 行動減少

## 現代ABAの特徴

- **機能的分析** が核: なぜその行動をするか
- **ポジティブ強化中心**: 罰ではなく望ましい行動の強化
- **自然な環境での学習**: DTT（個別試行訓練）だけでなく自然な機会の活用
- **個別化**: 本人ごとのカリキュラム

## 代表的プログラム

- **DTT**（Discrete Trial Training）: 個別試行訓練
- **NET**（Natural Environment Teaching）: 自然環境下の指導
- **Early Start Denver Model**: 乳幼児向け
- **Lovaas型**: 集中的早期介入

## 対象・エビデンス

- **エビデンスレベル: 高**
- 早期介入（2-5歳）の効果が最も研究されている

## 実施の要件

- 日本では **日本行動分析学会認定専門家** 制度
- 家族・支援者のトレーニングが実施成功の鍵

## 倫理的論点

- **嫌悪的手続き**（罰）の使用は厳しく制限
- 本人の意思・快感情の軽視への批判（自閉症自己擁護運動から）
- **新しい ABA** は本人の選択・自律を重視

## 計画相談での位置づけ

- **一つの有効な手段** として捉え、単独でなく組合せて
- 本人・家族の希望と合意の上で導入
- [[64_Methods/構造化支援]] との統合が実務的
