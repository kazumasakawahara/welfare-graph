# 認知行動療法（CBT）

## 概要

**考え方（認知）と行動の関係** に着目し、不適応的な認知パターンや行動習慣を変容させる心理療法。1960年代に Beck・Ellis らが体系化。

現在、多くの精神疾患で **第一選択の心理療法**。

## 基本モデル

```
出来事 → 自動思考（認知） → 感情・身体反応・行動
              ↑
         スキーマ（中核信念）
```

同じ出来事でも **認知** のフィルターで反応が変わる → 認知を修正すれば感情・行動も変わる。

## 主要技法

### 認知再構成
- **自動思考** の同定
- 根拠・反証の検討
- より適応的な認知の案出
- 記録表（コラム法）

### 行動活性化
- うつ病の第一技法
- 活動記録 → 快感情・達成感と結び付く活動を増やす
- 「やる気が出たら動く」ではなく「動くからやる気が出る」

### 曝露療法
- 不安症・PTSD で用いる
- 恐れる状況に段階的に近づく

### 問題解決療法
- 問題の特定
- 解決策案出
- 実行・評価

### マインドフルネス・アクセプタンス
- 第3世代CBT（MBCT、ACT等）
- 思考との距離を取る

## 対象・エビデンス

| 疾患 | エビデンス |
|---|---|
| **うつ病** | 最高水準、薬物療法と同等〜優位 |
| 不安症群・パニック症 | 最高水準 |
| PTSD | 高水準 |
| 強迫症 | 高水準 |
| **統合失調症**（CBTp） | 中等度、陽性症状・機能改善に |
| 摂食症 | 高水準 |
| 慢性疼痛 | 中等度 |

- **臨床心理士・公認心理師**
- 精神科医・心療内科医

## 日本での保険適用

- うつ病・不安症・強迫症・PTSD・過食症・神経性やせ症 に対し **医師実施の場合 保険適用**
- 16〜20回の構造化プログラム

## 発達障害・知的障害への適応

### 適応可能性

- **軽度知的障害 + うつ**: 認知概念を具体化して実施可
- **高機能ASD + 不安**: 可（特性への配慮必要）
- **中等度以上の知的障害**: 抽象的認知操作が難しい → 行動療法中心に

### 配慮点

- **視覚化**: コラム表をシートで具体化
- **具体例**: 抽象論を避け本人の経験に沿う
- **短時間・高頻度**: 疲労に配慮
- **家族協働**: セッション外の実践を家族がサポート

## 計画相談での活用

1. **二次障害としてのうつ・不安**: CBT実施機関への接続
2. **医療機関選定**: CBT実施可能な精神科・心療内科
3. **自立支援医療（精神通院）**: [[66_Services/精神通院医療]] との接続

## 禁忌・注意

- **急性期精神病状態**: まず薬物療法・安定化
- **自殺リスク高**: CBT単独では不十分、医療機関密接連携
- **重度認知機能低下**: 効果限定的、他アプローチ優先
